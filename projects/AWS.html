Building and Deploying an Application using Amazon Web Services

  • Tech Stack: Amazon EC2: Hosting application servers. Amazon S3: Storing static assets.Amazon CloudFront: Content delivery. Amazon Route 53: DNS management. Amazon RDS: Relational database service. AWS Lambda: Serverless functions.
  • Project URL: AWS Application

This Project is done as a part of AWS Mega Workshop.This project involves developing and deploying a scalable web application using key Amazon Web Services (AWS) technologies. The application will be hosted on Amazon EC2, with static assets stored in Amazon S3. Amazon CloudFront will be used for content delivery, while Amazon Route 53 will handle domain name system (DNS) management. The back-end will utilize Amazon RDS for database management and AWS Lambda for serverless computing tasks. Key Technologies: Amazon EC2: Hosting application servers. Amazon S3: Storing static assets.Amazon CloudFront: Content delivery. Amazon Route 53: DNS management. Amazon RDS: Relational database service. AWS Lambda: Serverless functions.
